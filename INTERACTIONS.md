# INTERACTIONS.md
## Brisart Cognitive Architecture – System Interaction Map

This file consolidates all published frameworks from the Brisart Research Archive and defines how they interact as a unified system.

---

# SYSTEM OVERVIEW

The architecture is organized into three layers:

## 1. Foundational Layer (Core Cognitive Construction)
- PFT — Perceptual framing (constructs experience)
- TFL — Memory feedback (injects past into present)
- TCRF — Temporal coherence (stabilizes flow of time)
- SST — Identity synchronization (stabilizes self-model)

## 2. Internal Systems Layer (Regulation & Modulation)
- IRE — Intention arbitration
- VRIF — Coherence and resonance modulation
- RIET — Identity-echo reactivation

## 3. Cross-Agent Layer
- PCT — Predictive alignment between interacting agents

---

# FRAMEWORK SUMMARIES + INTERACTIONS

---

## PFT — Perceptual Framing Theory
**Role:** Constructs the moment of experience.

**Key Function:**
- Converts sensory + memory + identity into a coherent frame

**Receives Input From:**
- TFL (memory traces)
- SST (identity bias)

**Feeds Into:**
- IRE (decision context)
- TCRF (temporal signature baseline)
- VRIF (elements to bind)
- RIET (environmental trigger context)
- PCT (observable output between agents)

---

## TFL — Temporal Feedback Loop
**Role:** Reactivates past experience into the present.

**Key Function:**
- Selects and feeds memory traces back into active perception

**Receives Input From:**
- PFT (current frame constraints)
- SST (identity weighting)

**Feeds Into:**
- PFT (frame shaping)
- IRE (emotional weighting)
- TCRF (temporal continuity alignment)
- VRIF (binding material)
- RIET (identity-echo source pool)
- PCT (predictive modeling inputs)

---

## TCRF — Temporal Coherence Resonance Framework
**Role:** Stabilizes experience across time.

**Key Function:**
- Aligns moment-to-moment temporal signatures

**Receives Input From:**
- PFT (current frame)
- TFL (past continuity)
- SST (identity consistency)
- IRE (intention trajectory)

**Feeds Into:**
- VRIF (stability basis for coherence)
- PCT (timing and latency consistency)

---

## SST — Simulation Synchronization Theory
**Role:** Stabilizes identity via feedback loops.

**Key Function:**
- Aligns internal self-simulation with emotional/social feedback

**Receives Input From:**
- PFT (context)
- TFL (identity-linked memory)
- External feedback signals

**Feeds Into:**
- TFL (memory prioritization)
- IRE (identity-aligned intention bias)
- TCRF (identity continuity)
- VRIF (resonance sensitivity)
- PCT (identity-level alignment across agents)

---

## IRE — Intent Resolution Engine
**Role:** Selects actions from competing intentions.

**Key Function:**
- Runs arbitration loop until one intention crosses resolution threshold

**Receives Input From:**
- PFT (context)
- TFL (emotional weight)
- SST (identity alignment)

**Feeds Into:**
- TCRF (execution continuity)
- PCT (behavioral output for prediction)

---

## VRIF — Vibrational Resonance Integration Framework
**Role:** Modulates coherence of experience.

**Key Function:**
- Controls binding strength, integration, and fragmentation

**Receives Input From:**
- PFT (frame elements)
- TFL (memory streams)
- SST (identity states)
- TCRF (temporal stability)

**Feeds Into:**
- RIET (activation thresholds)
- IRE (stability of decision space)
- PCT (alignment stability under high coherence)

---

## RIET — Residual Identity Echo Theory
**Role:** Reactivates identity states as perceptual simulations.

**Key Function:**
- Converts emotionally encoded identity traces into “echoes” under resonance conditions

**Receives Input From:**
- TFL (stored identity traces)
- PFT (environmental trigger context)
- VRIF (resonance thresholds)

**Feeds Into:**
- SST (identity integration pressure)
- IRE (interpretation of echo significance)
- PCT (non-required but adjacent interference or signal layer)

---

## PCT — Predictive Convergence Theory
**Role:** Cross-agent predictive alignment.

**Key Function:**
- Models how agents converge on each other's latent states under low-signal conditions

**Receives Input From:**
- PFT (observable outputs)
- TFL (predictive memory)
- SST (identity modeling)
- TCRF (timing stability)
- VRIF (coherence state)
- IRE (behavioral outputs)

**Produces:**
- Predictive alignment
- Error compression
- Reduced signaling dependence

---

# SYSTEM FLOW (HIGH-LEVEL)

1. PFT constructs the current perceptual frame  
2. TFL injects relevant past experience  
3. SST stabilizes identity within the frame  
4. IRE selects an action  
5. TCRF ensures continuity over time  
6. VRIF modulates coherence and binding  
7. RIET may inject identity-echoes under high resonance  
8. PCT operates across agents when interaction occurs  

---

# ONE-LINE DEFINITIONS

- PFT → Builds the moment  
- TFL → Feeds the past into the moment  
- SST → Stabilizes the self  
- IRE → Chooses the action  
- TCRF → Keeps time continuous  
- VRIF → Controls coherence  
- RIET → Reactivates identity echoes  
- PCT → Aligns minds across agents
